NovelShelf · Phase1 UIモック

読書に集中できる、静かな本棚。

広告なし・装飾を排したMaterial Design 3ベースのUI。以下6画面は縦横書き・ライト/ダーク・明朝/ゴシックの切り替えを含む読書体験の主要導線を示す。

実データではなくモックデータで構成。実装時は本文をOpenAPI（/chapters/{chapterId}/content）から取得する。

表示中のテーマはビューア設定（OS / 右上のテーマ切替）に追従します。本ページは静的モックのため操作はできません。実装時のコンポーネント分割方針は docs/phase1/directory-structure.md の frontend/src/components を参照。
